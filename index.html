Welcome to aquarius CMS.

aquarius is a web content management system (CMS) developed by aquaverde GmbH in Biel/Bienne - Switzerland.

aquarius-core

aquarius-core is only the core engine of the cms, without frontend elements. Please use an aquarius-package (per example aquarius-blank) for your first project.
